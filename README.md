# Retail Sales Analytics Dashboard

## Project Overview

This project is an end-to-end Retail Sales Analytics solution built using Python, SQL Server, SQL, and Power BI.

The project includes:

* Data loading and transformation using Python
* SQL Server database integration
* Business analysis using SQL queries
* Interactive Power BI dashboard creation
* KPI analysis using DAX measures
* Forecasting and trend analysis

---

## Tools & Technologies Used

* Python
* Pandas
* SQL Server
* SQL
* Power BI
* DAX

---

## Dataset

Global retail sales dataset containing:

* Orders
* Sales
* Profit
* Customers
* Categories
* Regions
* Shipping details

Time Period:
2011–2014

---

## Key Dashboard Features

* Total Sales KPI
* Total Profit KPI
* Profit Margin %
* Average Order Value
* Sales Trend Analysis
* Forecasting
* Regional Performance Analysis
* Top 10 Products by Sales
* Interactive Slicers
* Geographic Sales Map

---

## Project Workflow

1. Imported CSV dataset using Python
2. Loaded data into SQL Server database
3. Performed SQL-based business analysis
4. Connected SQL Server with Power BI
5. Created interactive dashboard and DAX measures
6. Added forecasting and business insights
